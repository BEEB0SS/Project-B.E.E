# Project-B.E.E
Modifying JPL's open-source robot, this project will focus on the development of a mobile robotic arm, similar to DUM-E in Iron Man. This project is hosted by NCSSM-Morganton and will span multiple years across multiple students. The purpose of this project is to be used to develop the engineering skills of students in mechanical, electrical, software, and AI engineering. This project is being sponsered by Mr. Hilton from NCSSM-Morganton. Currently, this project is being led by Pratham Madaram '24 from 2022-2024.
